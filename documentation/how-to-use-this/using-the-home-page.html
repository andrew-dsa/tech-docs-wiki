Using the home page

If a Github wiki contains a page called “Home” then that page will displayed as the front page of the wiki.

This functionality is mirrored on the generated static site. If a page called “Home” exists in the wiki it will be used as the front page for the static site and will not exist under the name “Home”.

The title of the front page can be customised by manually specifying an H1 level heading at the top of the page using a single #:
